## Workflow Examples

### Podcast Launch
1. **Plan content** → podcast-production (structure)
2. **Record interviews** → remote-interview (setup) + podcast-interview (technique)
3. **Edit audio** → audio-editing (processing)
4. **Optimize for SEO** → podcast-seo (transcripts, show notes)
5. **Repurpose content** → transcription-to-content (blog, social)

### Brand Sound Development
1. **Define identity** → sonic-branding (strategy)
2. **Create audio logo** → audio-logo-design (3-5 second identifier)
3. **Generate variations** → voice-design (if AI voice needed)
4. **Localize** → voice-localization (multilingual markets)

### Video Course Production
1. **Record voiceover** → voiceover-direction (performance)
2. **Edit audio** → audio-editing (quality)
3. **Add sound design** → ux-sound-design (feedback sounds)
4. **Automate processing** → pydub-automation (batch export)
